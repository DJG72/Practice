## About This Repository

This repository contains all hands-on exercises, assignments, projects, and learning materials completed as part of the **Cognizant Digital Nurture 5.0 Program** under the **.NET Full Stack Engineer (FSE)** track.

The program focuses on software engineering fundamentals, design patterns, data structures and algorithms, database management, backend development, frontend development, testing frameworks, cloud-native technologies, DevOps practices, and Generative AI concepts.
